CRUD operations required in hostel room allotment system:

1. Create
  Admin:
  - Register student in hostel
  - Create a new room
  Student:
  - Register student in hostel

2. Read
  Admin:
  - Read all rooms
  - Read all students
  Student:
  - Read own room
  - Read all rooms (without students sensitive data)
  
3. Update
  Admin:
  - Change a student room
  - Allot a room
  - Deallot a room
  Student:
  - Allot a room
  - Leave a room
  - Change a room
  - Update a student
  
4. Delete
  Admin:
  - Delete a student
  - Delete a room
  Student:
  - Delete a student
